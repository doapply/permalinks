# permalinks [![NPM version](https://badge.fury.io/js/permalinks.png)](http://badge.fury.io/js/permalinks)

> Easily add permalinks to any node.js project.

## Install
Install with [npm](npmjs.org):

```bash
npm i permalinks --save-dev
```
